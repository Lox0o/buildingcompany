**File naming:** Use the worker's first name, e.g. `james.jpg`, `mike.jpg`

**Recommended size:** 400x400px (square) or 400x600px (portrait)

**What to use:**
- Job site photos (not studio portraits)
- Natural, authentic shots
- Shows them working or with tools/project
- Can be casual/relaxed
