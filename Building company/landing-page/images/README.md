# Image Folders

Organize your images here:

- **hero/** - Hero section background image/video
- **portfolio/** - Project photos (one folder per project, or all together)
- **about/** - Photo of Ed on site
- **testimonials/** - Worker photos (when you have testimonials)

## Image Requirements

- **Hero**: Full-width image, ideally 1920x1080px or larger. Can be a video too.
- **Portfolio**: High-resolution photos (min 1200px wide). Show before/during/after.
- **About Ed**: Portrait or job-site photo, ideally 800x1000px or similar.
- **Testimonials**: Square or portrait photos, 400x400px minimum.

## File Formats

- Use JPEG for photos
- Use WebP if possible (better compression)
- Keep file sizes reasonable (under 500KB per image when possible)
