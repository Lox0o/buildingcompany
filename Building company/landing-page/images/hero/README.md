# Hero Images

Drop your hero background image here.

**File name:** `hero-bg.jpg` (or `.png`, `.webp`)

**Recommended size:** 1920x1080px or larger (16:9 aspect ratio)

**What to use:**
- Job site photo showing quality work
- Action shot of carpentry in progress
- Finished project exterior/interior
- Team working together

**Tips:**
- Use a high-quality, well-lit photo
- Avoid busy backgrounds (text will overlay)
- The image will be darkened with an overlay for text readability
